Hölder's Inequality

Theorem

Let \(f \in \mathbb{L}^p(\mathbb{R})\) and \(g \in \mathbb{L}^q(\mathbb{R})\) where \(\frac{1}{p}+\frac{1}{q}=1\) for \(p,q \geq 1\). Then, \[ \|fg\|_1 \leq \|f\|_p\|g\|_q.\]

Proof

First we can see that for \(a,b > 0\) we have \[ ab \leq \dfrac{a^p}{p} + \dfrac{b^q}{q} \tag{1} \label{eq:ineq}. \] This can be shown via Jensen's inequality. We see \[ \log (ab ) = \dfrac{1}{p} \log (a^p) + \dfrac{1}{q} \log (a^q) \] \[ \leq \log \left (\dfrac{a^p}{p} + \dfrac{a^q}{q} \right ) . \] This gives the inequality \((\ref{eq:ineq})\). Now, we prove the result for \(f,g\) such that \(\|f\|_p = \|g\|_q = 1\).
\[\|fg\|_1 = \int_{\mathbb{R}} |f(x)g(x)| dx \leq \dfrac{1}{p} \int_{\mathbb{R}} |f(x)|^p dx + \dfrac{1}{q} \int_{\mathbb{R}} |g(x)|^q dx \]
\[\|fg\|_1 = \int_{\mathbb{R}} |f(x)g(x)| dx \] \[ \leq \dfrac{1}{p} \int_{\mathbb{R}} |f(x)|^p dx + \dfrac{1}{q} \int_{\mathbb{R}} |g(x)|^q dx \]
\[ \leq \dfrac{1}{p} + \dfrac{1}{q} = 1 .\] Thus we can see that in general give arbitrary \(f\) and \(g\) \[\left \| \frac{f}{\|f\|_p} \frac{g}{\|g\|_q} \right \|_1 \leq 1 ,\] which gives \[\left \| fg \right \|_1 \leq \|f\|_p \|g\|_q. \]
