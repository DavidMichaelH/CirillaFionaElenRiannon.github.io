Moment Generating Function Bound Implies Zero Mean

Theorem

Suppose that there is \(c>0\) so that for all \(t \in \mathbb{R}\) we have that \[ \mathbb{E} \exp( t X) \leq \exp( c t^2) .\] Show that \(\mathbb{E} X = 0\).

Proof

Suppose the bound holds. Then \[ \mathbb{E} \exp( t X) = 1 + t \mathbb{E} X + t^2 \sum_{k=2}^{\infty} \dfrac{t^{k-2}}{k!} \mathbb{E} X^k \] \[ \leq 1 + 0 + t^2 \sum_{k=2}^{\infty} \dfrac{c^k t^{2(k-1)}}{k!} = \exp( c t^2) . \] Thus \[ \mathbb{E} X + t \sum_{k=2}^{\infty} \dfrac{t^{k-2}}{k!} \mathbb{E} X^k \leq t \sum_{k=2}^{\infty} \dfrac{c^k t^{2(k-1)}}{k!} . \] Ltting \(t \to 0\) we have that \(\mathbb{E} X \leq 0\). Now we can rerun the same argument after swapping \(t\) with \(-t\). This gives \[ \mathbb{E} \exp( -t X) = 1 - t \mathbb{E} X + t^2 \sum_{k=2}^{\infty} \dfrac{(-1)^kt^{k-2}}{k!} \mathbb{E} X^k \] \[ \leq 1 + 0 + t^2 \sum_{k=2}^{\infty} \dfrac{c^k t^{2(k-1)}}{k!} = \exp( c t^2) . \] \[ - \mathbb{E} X + t \sum_{k=2}^{\infty} \dfrac{t^{k-2}}{k!} \mathbb{E} X^k \leq t \sum_{k=2}^{\infty} \dfrac{c^k t^{2(k-1)}}{k!} . \] Again letting \(t \to 0\) we have that \(-\mathbb{E} X \leq 0\) or indeed \(\mathbb{E} X \geq 0\) and so \(\mathbb{E} X = 0\).
