  • YouTube Comment Sentiment Analyzer [Application] [Repository]
  • Flask-based web app designed to analyze the sentiment of comments on YouTube videos.
  • The app is built with Python, Flask, HTML, CSS, and JavaScript and integrates with the YouTube API to fetch comments from a specified video.
  • It leverages the Hugging Face API pretrained models for sentiment analysis.
  • The user interface is intuitive, allowing users to input a YouTube video URL and view the analyzed comments, their sentiment labels, and scores.
  • Displays summary statistics, including the number of positive and negative comments, their respective percentages, and an overall sentiment score.
  • Deployed the app on GCP for scalability and reliability.
